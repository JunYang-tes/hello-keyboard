DIY 小红点键盘

## 特性
固件设计为vimer优化。
- 小红点（鼠标）
- Caps位置单独按下并释放等同与Esc，Caps 按下期间有其它键按下/释放 等同于Ctrl+那个键
- Fn + h/j/k/l 等于方向键，Fn + 数字 等于 F1 ~ F12
